# 급여납치 플랫폼 모듈 경계 및 의존성 정의서 최종본

## 허용 의존성

```text
apps/mobile ─┬─> packages/api-contract
             ├─> packages/types
             ├─> packages/ui
             ├─> packages/utils
             └─> packages/security

apps/admin ──┬─> packages/api-contract
             ├─> packages/types
             ├─> packages/ui
             └─> packages/utils

services/api ─┬─> packages/api-contract
              ├─> packages/types
              ├─> packages/db
              ├─> packages/security
              └─> packages/utils

services/scheduler ─┬─> packages/db
                    ├─> packages/types
                    ├─> packages/security
                    └─> services/notifications

services/notifications ─┬─> packages/db
                        ├─> packages/types
                        └─> packages/security
```

## 금지 의존성

- `packages/`는 `apps/`에 의존하지 않는다.
- `packages/`는 `services/`에 의존하지 않는다.
- `apps/mobile/`은 DB에 직접 접근하지 않는다.
- `apps/admin/`은 DB에 직접 접근하지 않는다.
- `services/api/`는 모바일 UI 컴포넌트를 참조하지 않는다.
- `database/`는 앱 코드에 의존하지 않는다.
- `infra/`는 비즈니스 로직을 포함하지 않는다.

## 도메인 경계

| 도메인       | 주요 책임                  | 금지 사항                      |
| ------------ | -------------------------- | ------------------------------ |
| Auth         | 로그인, 토큰, 권한         | 급여 계산 포함 금지            |
| Salary       | 급여일, 수령금액, 납치금액 | 커뮤니티 로직 포함 금지        |
| Budget       | 일일 예산, 남은금액        | 인증 로직 포함 금지            |
| Expense      | 고정/변동지출              | 게시글 상태 변경 금지          |
| Growth       | 독서/뉴스/영어/건강 미션   | 광고 타겟팅 로직 포함 금지     |
| Community    | 게시글, 댓글, 신고         | 급여 원본 데이터 조회 금지     |
| Notification | 알림 생성/발송             | DB 스키마 변경 금지            |
| Admin        | 운영 관리                  | 사용자 비밀번호 원문 접근 금지 |
