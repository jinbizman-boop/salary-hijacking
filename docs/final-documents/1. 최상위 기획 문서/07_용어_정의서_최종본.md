# 용어 정의서 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼에서 사용하는 핵심 용어, 기능 용어, 데이터 용어, 정책 용어, 상태값 용어를 최종 확정하기 위한 기준 문서다. 본 문서는 기획, 디자인, 개발, QA, 운영, 고객지원 전 영역에서 동일한 의미로 용어를 사용하기 위해 작성한다.

본 문서는 문서상·이론상 최종본이며, 급여납치 플랫폼에서 사용하는 용어 정의에 대해 추가 정리 작업이 필요 없도록 완결한다.

## 2. 용어 사용 원칙

1. 문서와 화면에서는 본 정의서의 용어를 우선 사용한다.
2. 같은 의미의 다른 표현은 혼용하지 않는다.
3. 금액 관련 용어는 계산식과 연결해 사용한다.
4. 사용자에게 노출되는 용어는 쉽고 직관적으로 표현한다.
5. 개발 데이터 필드명은 영문을 사용하되, 의미는 본 정의서를 따른다.
6. 법적·정책상 혼동될 수 있는 용어는 설명 문구를 함께 제공한다.

## 3. 핵심 서비스 용어

| 용어             | 정의                                                          | 사용 위치              | 비고                                    |
| ---------------- | ------------------------------------------------------------- | ---------------------- | --------------------------------------- |
| 급여납치         | 월급을 사라지기 전에 먼저 확보하고 관리한다는 의미의 서비스명 | 앱명, 로고, 온보딩     | 부정적 의미가 아닌 자기관리 의미로 설명 |
| Salary Hijacking | 급여납치의 영문 서비스명                                      | 로고, 스플래시, 브랜딩 | 영문명 고정                             |
| 납치             | 월급 중 사용자가 지켜낸 금액을 확보하는 행위                  | 브랜드 문구, 설명      | 실제 불법행위 의미 아님                 |
| 납치금액         | 수령금액에서 지출금액을 제외해 사용자가 지켜낸 금액           | 홈, 계획, 마이페이지   | 핵심 성과 지표                          |
| 누적 납치금액    | 월별 납치금액이 누적된 총액                                   | 홈, 마이페이지         | 사용자 성취 지표                        |
| 목표 납치금액    | 사용자가 달성하고자 설정한 목표 금액                          | 계획, 달성률           | 월별 또는 누적 기준 가능                |
| 목표 달성률      | 목표 납치금액 대비 누적 납치금액의 비율                       | 계획, 홈               | 백분율 표시                             |
| 급여 자기관리    | 급여, 소비, 저축, 자기계발을 함께 관리하는 행위               | 서비스 설명            | 서비스 핵심 방향                        |

## 4. 금액 관련 용어

| 용어               | 정의                                                     | 계산/처리 기준                   |
| ------------------ | -------------------------------------------------------- | -------------------------------- |
| 수령금액           | 사용자가 입력한 실제 또는 예정 월급 금액                 | 원 단위 정수                     |
| 수령 예정 급여     | 다음 급여일에 받을 것으로 예상하는 급여                  | 급여계획 입력값                  |
| 실제 수령 급여     | 실제 지급받은 급여                                       | 월 마감 시 확정 가능             |
| 지출금액           | 고정지출과 변동지출을 합산한 금액                        | 고정지출 + 변동지출              |
| 지출 예정 금액     | 월별로 예상되는 지출 총액                                | 계획 화면 입력값                 |
| 고정지출           | 매월 반복적으로 발생하는 지출                            | 구독료, 대출, 통신비, 보험료 등  |
| 변동지출           | 사용자가 당일 또는 특정 시점에 직접 기록하는 비정기 지출 | 식비, 교통비, 쇼핑 등            |
| 고정저축           | 매월 반복적으로 저축·투자·청약 등에 배정하는 금액        | 납치금액 계산에 포함 가능        |
| 생활비             | 일정 기간 동안 사용할 소비 예산                          | 월별 생활비와 일일 예산으로 분리 |
| 일일 설정금액      | 하루 동안 사용하기로 정한 예산                           | 사용자가 설정                    |
| 오늘 사용금액      | 오늘 기록된 변동지출 합계                                | 날짜 기준 합산                   |
| 오늘 남은금액      | 일일 설정금액에서 오늘 사용금액을 제외한 금액            | 일일 설정금액 - 오늘 사용금액    |
| 초과 사용금액      | 오늘 사용금액이 일일 설정금액을 초과한 금액              | 음수 남은금액의 절대값           |
| 월별 확정 납치금액 | 월 마감 후 실제 수령금액에서 실제 지출금액을 제외한 금액 | 월 단위 확정                     |

## 5. 계산 용어

| 용어          | 정의                                  | 산식                                |
| ------------- | ------------------------------------- | ----------------------------------- |
| 예상 납치금액 | 계획 기준으로 예상되는 확보 금액      | 수령 예정 급여 - 지출 예정 금액     |
| 오늘 남은금액 | 오늘 아직 사용할 수 있는 예산         | 일일 설정금액 - 오늘 사용금액       |
| 총 지출금액   | 일정 기간 동안의 지출 합계            | 고정지출 + 변동지출                 |
| 누적 납치금액 | 월별 확정 납치금액의 누적             | 기존 누적 + 월별 확정 납치금액      |
| 목표 달성률   | 목표 대비 성과 비율                   | 누적 납치금액 / 목표 납치금액 × 100 |
| 예산 초과     | 사용금액이 설정금액보다 큰 상태       | 오늘 사용금액 > 일일 설정금액       |
| 월 마감       | 해당 월의 급여와 지출을 확정하는 처리 | 월별 성과 확정 시점                 |
