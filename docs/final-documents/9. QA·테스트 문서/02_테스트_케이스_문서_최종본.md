# 02. 테스트 케이스 문서 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼의 기능별 테스트 케이스를 확정한다. 모든 테스트 케이스는 테스트 ID, 우선순위, 사전조건, 절차, 기대결과, 판정 기준을 포함한다.

## 2. 공통 판정 기준

| 판정    | 의미                                 |
| ------- | ------------------------------------ |
| PASS    | 기대결과와 실제결과가 완전히 일치    |
| FAIL    | 기대결과와 실제결과가 불일치         |
| BLOCKED | 선행 결함 또는 환경 문제로 수행 불가 |
| N/A     | 해당 빌드 또는 범위에 적용되지 않음  |
