# 06. 디바이스 테스트 매트릭스 최종본

## 1. 문서 목적

본 문서는 급여납치 모바일 앱의 iOS/Android 디바이스, 화면 크기, 해상도, OS 버전, 권한, 네트워크, 다크모드, 접근성 대응 테스트 기준을 확정한다.

## 2. 테스트 기준 원칙

1. 모바일 앱은 작은 화면에서도 핵심 금액 카드가 잘리지 않아야 한다.
2. iOS와 Android 모두 safe area, notch, gesture navigation, keyboard overlay를 고려한다.
3. Android는 다양한 화면 크기와 픽셀 밀도 대응을 검증한다.
4. iOS는 최신 3개 주요 버전과 대표 화면 크기를 검증한다.
5. 다크모드 지원 여부가 MVP 범위에 포함되지 않아도 최소한 깨짐 여부는 확인한다.
