# 08. 보안 테스트 체크리스트 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼의 인증 우회, 권한 오류, SQL Injection, XSS, 파일 업로드 악성 처리, API 보안, 모바일 앱 보안, 개인정보 보호 검증 항목을 확정한다.

## 2. 보안 테스트 기준

급여납치 플랫폼은 급여, 대출, 저축, 소비 내역을 다루므로 일반 커뮤니티 앱보다 높은 보안 기준을 적용한다. 보안 테스트는 OWASP MASTG, OWASP API Security Top 10, OWASP Web Top 10 관점의 항목을 포함한다.
