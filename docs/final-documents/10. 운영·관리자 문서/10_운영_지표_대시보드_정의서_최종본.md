---

| 항목        | 내용                                                                                   |
| ----------- | -------------------------------------------------------------------------------------- |
| 프로젝트    | 급여납치 Salary Hijacking 플랫폼                                                       |
| 문서 상태   | 문서상·이론상 최종본                                                                   |
| 기준일      | 2026-06-15                                                                             |
| 적용 범위   | 모바일 앱, API 서버, Neon DB, Cloudflare, GitHub 기반 운영 환경                        |

## 1. 목적

본 문서는 급여납치 플랫폼의 운영 상태를 확인하기 위한 대시보드 지표를 정의한다. 대시보드는 사용자 활성도, 급여관리 핵심 행동, 레벨업, 커뮤니티, 알림, 광고/제휴, CS, 장애, 보안/정책 지표를 한 곳에서 확인하도록 설계한다.

## 2. 대시보드 구성

| 대시보드                   | 목적                              | 사용자                         |
| -------------------------- | --------------------------------- | ------------------------------ |
| Executive Dashboard        | 전체 서비스 상태 요약             | 대표, 제품 책임자, 운영 책임자 |
| Growth Dashboard           | 가입, 활성, 리텐션 분석           | 제품/마케팅                    |
| Payroll Dashboard          | 급여계획, 지출기록, 납치금액 분석 | 제품/운영                      |
| Level Up Dashboard         | 콘텐츠/미션 성과 분석             | 콘텐츠 운영                    |
| Community Dashboard        | 게시글, 댓글, 신고, 제재 분석     | 커뮤니티 운영                  |
| Notification Dashboard     | 푸시/알림 성과 분석               | 운영/백엔드                    |
| Ads Dashboard              | 광고 노출, 클릭, 신고, 정산 분석  | 광고/제휴 운영                 |
| CS Dashboard               | 문의량, SLA, 만족도 분석          | CS 운영                        |
| Reliability Dashboard      | 장애, 성능, 오류율 분석           | 기술/운영                      |
| Privacy/Security Dashboard | 접근로그, 권한, 정책 리스크 분석  | 보안/개인정보 담당             |

## 3. 핵심 KPI

| KPI              | 정의                                | 산식                                | 목표 방향 |
| ---------------- | ----------------------------------- | ----------------------------------- | --------- |
| DAU              | 일간 활성 사용자                    | 일간 앱 방문 고유 사용자 수         | 증가      |
| WAU              | 주간 활성 사용자                    | 7일간 앱 방문 고유 사용자 수        | 증가      |
| MAU              | 월간 활성 사용자                    | 30일간 앱 방문 고유 사용자 수       | 증가      |
| 신규 가입자 수   | 신규 계정 생성                      | 일/주/월 가입 수                    | 증가      |
| 급여계획 등록률  | 가입 후 급여계획 등록 비율          | 급여계획 등록 사용자 / 신규 가입자  | 증가      |
| 일일 지출 기록률 | 하루 1건 이상 지출 기록 사용자 비율 | 지출 기록 사용자 / DAU              | 증가      |
| 목표 설정률      | 목표 납치금액 설정 사용자 비율      | 목표 설정 사용자 / 활성 사용자      | 증가      |
| 목표 달성률      | 목표 납치금액 달성 사용자 비율      | 목표 달성 사용자 / 목표 설정 사용자 | 증가      |
| 평균 납치금액    | 사용자별 월 확정 납치금액 평균      | 월 확정 납치금액 합계 / 사용자 수   | 증가      |
| 레벨업 완료율    | 미션 시작 대비 완료 비율            | 완료 수 / 시작 수                   | 증가      |
| 커뮤니티 참여율  | 게시/댓글/좋아요 참여 사용자 비율   | 참여 사용자 / 활성 사용자           | 증가      |
| 알림 클릭률      | 알림 수신 대비 클릭 비율            | 클릭 / 수신                         | 증가      |
| CS SLA 준수율    | SLA 내 처리 비율                    | SLA 내 처리 / 전체 티켓             | 증가      |
| 장애 MTTR        | 평균 복구 시간                      | 복구 완료-장애 발생 평균            | 감소      |

## 5. 지표별 상세 정의

### 5.1 사용자 활성 지표

| 지표        | 계산 기준                      | 집계 주기 | 필터                |
| ----------- | ------------------------------ | --------- | ------------------- |
| DAU         | app_open 고유 사용자           | 일간      | OS, 버전, 가입 경로 |
| WAU         | 최근 7일 app_open 고유 사용자  | 일간      | OS, 버전            |
| MAU         | 최근 30일 app_open 고유 사용자 | 일간      | OS, 버전            |
| 신규 가입자 | sign_up_completed              | 일간      | provider            |
| 탈퇴자      | account_deleted                | 일간      | 탈퇴 사유           |

### 5.2 급여관리 지표

| 지표                   | 산식                                      | 주의                                 |
| ---------------------- | ----------------------------------------- | ------------------------------------ |
| 급여계획 등록률        | payroll_plan_created 사용자 / 신규 가입자 | 중복 생성 제외                       |
| 지출 기록 사용자 수    | variable_expense_created 고유 사용자      | 삭제된 지출 제외 기준 별도 표시      |
| 평균 일일 지출 기록 수 | 지출 기록 수 / 기록 사용자                | 봇/테스트 제외                       |
| 평균 납치금액          | 월 확정 납치금액 합계 / 확정 사용자       | 원문 금액은 대시보드에 비식별/범주화 |
| 예산 초과율            | 일일 예산 초과 일수 / 예산 설정 일수      | 음수 잔액 기준                       |

### 5.3 레벨업 지표

| 지표              | 산식                                        |
| ----------------- | ------------------------------------------- |
| 미션 시작률       | growth_task_started 사용자 / DAU            |
| 미션 완료율       | growth_task_completed / growth_task_started |
| 카테고리별 완료율 | taskType별 완료 / 시작                      |
| 레벨 상승 사용자  | level_up 발생 사용자 수                     |
| 콘텐츠 신고율     | 콘텐츠 신고 수 / 콘텐츠 노출 수             |

### 5.4 커뮤니티 지표

| 지표            | 산식                               |
| --------------- | ---------------------------------- |
| 게시글 수       | post_created 수                    |
| 댓글 수         | comment_created 수                 |
| 좋아요 수       | post_liked 수                      |
| 커뮤니티 참여율 | 게시/댓글/좋아요 사용자 / DAU      |
| 신고율          | report_created / 게시글+댓글 수    |
| 제재율          | 제재 사용자 / 커뮤니티 참여 사용자 |

### 5.5 알림 지표

| 지표           | 산식                             |
| -------------- | -------------------------------- |
| 알림 발송 수   | notification_sent 수             |
| 알림 도달률    | delivered / sent                 |
| 알림 클릭률    | clicked / delivered              |
| 알림 해제율    | 알림 OFF 사용자 / 알림 ON 사용자 |
| 중복 발송 건수 | 동일 키 중복 발송 로그 수        |

### 5.6 광고/제휴 지표

| 지표         | 산식                       |
| ------------ | -------------------------- |
| 광고 노출 수 | ad_impression 수           |
| 광고 클릭률  | ad_click / ad_impression   |
| 랜딩 성공률  | landing_success / ad_click |
| 광고 신고율  | ad_report / ad_impression  |
| 캠페인 매출  | 계약 방식별 정산 금액      |

### 5.7 CS/운영 지표

| 지표                 | 산식                         |
| -------------------- | ---------------------------- |
| 문의 접수 수         | cs_ticket_created 수         |
| 카테고리별 문의 비율 | 카테고리 문의 / 전체 문의    |
| 1차 응답 시간        | 최초 답변-접수 시간          |
| SLA 준수율           | SLA 내 처리 / 전체 처리      |
| 재문의율             | 동일 이슈 재오픈 / 종료 문의 |

### 5.8 안정성 지표

| 지표         | 산식/기준                       |
| ------------ | ------------------------------- |
| API 오류율   | 5xx 응답 / 전체 응답            |
| p95 응답시간 | API별 95퍼센타일 응답시간       |
| 앱 크래시율  | 크래시 세션 / 전체 세션         |
| 배치 실패율  | 실패 배치 / 전체 배치           |
| MTTR         | 복구 완료 시각 - 장애 발생 시각 |

## 9. 운영 리포트 양식

### 9.1 일간 리포트

- DAU/신규 가입자/탈퇴자
- 급여계획 등록률
- 지출 기록 사용자 수
- 주요 장애/오류
- CS 문의량 및 주요 이슈
- 커뮤니티 신고/제재 현황

### 9.2 주간 리포트

- WAU/리텐션
- 급여관리 핵심 퍼널
- 레벨업 콘텐츠 성과
- 커뮤니티 활성도
- 이벤트 성과
- 광고/제휴 성과
- 장애/성능 요약

### 9.3 월간 리포트

- MAU/성장률
- 평균 납치금액 및 목표 달성률
- 수익화 성과
- 고객 문의 개선 과제
- 정책/보안 이슈
- 다음 달 운영 액션

## 10. 완료 선언

본 문서는 급여납치 운영 지표 대시보드의 문서상·이론상 최종 기준이다. 본 문서의 KPI, 이벤트 로그, 지표 산식, 권한, 임계값, 리포트 기준을 충족하면 운영 지표 체계는 최종 완료 상태로 판정한다.
