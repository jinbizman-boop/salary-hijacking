> 문서 상태: 최종본  
> 적용 서비스: 급여납치(Salary Hijacking) 모바일 플랫폼  
> 적용 범위: 급여관리 · 가계부 · 자기계발 레벨업 · 커뮤니티 · 알림 · 광고/제휴 · 운영 데이터  
> 기준 DB: Neon DB / PostgreSQL  
> 기준일: 2026-06-15  
> 원칙: 본 문서는 문서상·이론상 추가 작업 없이 구현 기준으로 사용할 수 있도록 완결된 기준값, 필드, 관계, 정책, 예외를 포함한다.

## 1. 문서 목적

본 문서는 급여납치 플랫폼의 개발, QA, 데모, API 테스트를 위한 표준 샘플 데이터를 제공한다. 샘플 데이터는 앱 기획안의 대표 화면인 급여 홈, 계획, LV UP, 커뮤니티, 글쓰기, 마이페이지, 알림을 모두 검증할 수 있도록 구성한다.

## 6. 샘플 데이터 검증 쿼리

### 6.1 홈 화면 금액 조회

```sql
SELECT
  pp.expected_salary_amount,
  pp.expected_expense_amount,
  pp.expected_hijack_amount,
  db.daily_limit_amount,
  db.used_amount,
  db.remaining_amount,
  db.over_amount,
  db.status
FROM payroll_plans pp
JOIN daily_budgets db ON db.user_id = pp.user_id
WHERE pp.user_id = '11111111-1111-1111-1111-111111111111'
  AND pp.year_month = '2026-06'
  AND db.budget_date = '2026-06-15';
```

### 6.2 변동지출 합계 검증

```sql
SELECT COALESCE(SUM(amount), 0) AS used_amount
FROM variable_expenses
WHERE daily_budget_id = '33333333-3333-3333-3333-111111111111'
  AND status = 'ACTIVE';
```

기대 결과: `13000`

### 6.3 커뮤니티 인기글 조회

```sql
SELECT board_type, title, view_count, like_count, comment_count
FROM community_posts
WHERE status = 'PUBLISHED'
ORDER BY like_count DESC, comment_count DESC, view_count DESC
LIMIT 10;
```

### 6.4 미읽음 알림 조회

```sql
SELECT type, title, body, created_at
FROM notifications
WHERE user_id = '11111111-1111-1111-1111-111111111111'
  AND status IN ('CREATED','SCHEDULED','SENT')
  AND read_at IS NULL
ORDER BY created_at DESC;
```

## 7. QA 시나리오별 기대값

| 시나리오    | 입력/상태                     | 기대값                                                   |
| ----------- | ----------------------------- | -------------------------------------------------------- |
| 홈 진입     | 샘플 사용자 로그인            | 수령금액 2,700,000원 표시                                |
| 일일예산    | 20,000원 한도 / 13,000원 사용 | 남은금액 7,000원 표시                                    |
| 지출 추가   | 15,000원 추가                 | used=28,000 / remaining=0 / over=8,000 / status=EXCEEDED |
| 계획 저장   | 급여 3,000,000 / 지출 700,000 | 예상 납치금액 2,300,000                                  |
| 레벨업 완료 | 경험치 70 추가                | current_exp 증가, 100 이상이면 level 증가                |
| 글쓰기      | 제목/본문/게시판 입력         | PUBLISHED 게시글 생성                                    |
| 익명 글     | is_anonymous=true             | 작성자명 미표시                                          |
| 알림 읽음   | read_at 저장                  | 새 알림 배지에서 제외                                    |

## 8. 최종 적용 기준

이 샘플 데이터는 로컬 개발, 스테이징, QA 자동화, 화면 시연에서 동일하게 사용한다. 운영 DB에는 본 샘플 데이터를 삽입하지 않는다. 샘플 데이터는 실제 개인정보가 아니며, 외부 서비스 계정이나 실제 결제 정보와 연결하지 않는다.
