## 8. 환경변수 보안 규칙

- `.env.local`은 Git에 커밋하지 않는다.
- `.env.example`에는 실제 secret 값을 넣지 않는다.
- 운영 secret은 GitHub Secrets 또는 Cloudflare Secret에 저장한다.
- DB URL, OAuth secret, FCM private key는 채팅/이슈/PR 본문에 노출하지 않는다.
- 유출 의심 시 즉시 폐기·재발급한다.

## 12. DB 마이그레이션

```bash
pnpm db:migrate
pnpm db:seed
pnpm db:studio
```

마이그레이션 원칙:

- DB 변경은 반드시 migration 파일로 반영한다.
- 운영 적용 전 dev/staging에서 검증한다.
- 롤백 또는 보상 절차를 문서화한다.

## 13. 로컬 개발 계정

| 계정                                  | 용도                 |
| ------------------------------------- | -------------------- |
| `user.demo@salary-hijacking.local`    | 일반 사용자          |
| `creator.demo@salary-hijacking.local` | 커뮤니티 활동 사용자 |
| `admin.demo@salary-hijacking.local`   | 관리자               |
| `blocked.demo@salary-hijacking.local` | 제재 사용자 테스트   |

비밀번호는 `.env.local` 또는 seed 문서에서 관리하며 실제 운영 계정과 절대 혼용하지 않는다.

## 16. 문제 해결

| 문제              | 조치                                 |
| ----------------- | ------------------------------------ |
| 패키지 설치 실패  | `pnpm store prune` 후 재설치         |
| 환경변수 오류     | `.env.example`과 비교                |
| DB 연결 실패      | Neon URL, SSL, IP 제한 확인          |
| iOS 빌드 실패     | Xcode 버전, pod 설치 상태 확인       |
| Android 빌드 실패 | SDK, Gradle, emulator 상태 확인      |
| API 401           | access token 만료, refresh flow 확인 |
| 푸시 실패         | FCM key, device token 확인           |
