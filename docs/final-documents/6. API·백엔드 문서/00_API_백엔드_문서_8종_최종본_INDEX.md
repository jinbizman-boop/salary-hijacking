## 3. 적용 기준

| 구분       | 기준                                                |
| ---------- | --------------------------------------------------- |
| API 버전   | `/api/v1`                                           |
| 관리자 API | `/admin/api/v1`                                     |

## 4. 최종 완료 선언

본 API·백엔드 문서 8종은 급여납치 플랫폼의 모바일 앱 구현, 관리자 운영, 데이터 저장, 파일 업로드, 푸시 알림, 검색, 배치 자동화에 필요한 백엔드 기준을 모두 포함한다. 따라서 본 문서 세트는 문서상·이론상 더 이상 수행할 작업이 없는 최종본으로 확정한다.
