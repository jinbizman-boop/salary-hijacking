# 01. 급여납치 출시 체크리스트 최종본

> 문서 상태: 최종본  
> 적용 범위: 급여납치 모바일 애플리케이션 iOS/Android 정식 출시  
> 작성 기준일: 2026-06-15  
> 기준 시간대: Asia/Seoul  
> 문서 원칙: 본 문서는 문서상·이론상 추가 기획 작업이 필요 없도록 출시 준비, 검수, 등록, 마케팅, 버전, 롤백 기준을 완결형으로 정의한다.

---

## 1. 문서 목적

본 문서는 급여납치 모바일 앱을 App Store와 Google Play에 제출하기 전 기능, 디자인, 데이터, 보안, 개인정보, QA, 운영, 스토어 등록, 배포, 롤백 준비 상태를 최종 확인하기 위한 출시 승인 체크리스트다. 본 문서의 모든 필수 항목이 `완료` 상태가 되기 전에는 정식 배포를 진행하지 않는다.

## 12. NO-GO 자동 조건

다음 항목 중 하나라도 발생하면 배포는 즉시 중단한다.

1. 급여, 대출, 저축, 소비 내역이 타 사용자에게 노출되는 오류.
2. 로그인 우회 또는 권한 우회 가능성.
3. 납치금액, 일일 남은금액, 목표 달성률 계산 오류.
4. 앱 실행 불가, 홈 화면 진입 불가, 저장 기능 불가.
5. 개인정보처리방침 URL 미제공 또는 실제 수집 항목과 스토어 고지 불일치.
6. 스토어 스크린샷/설명이 실제 기능과 다름.
7. 결제, 광고, 제휴, 이벤트 보상 문구가 실제 제공 조건과 불일치.
8. 심사 계정으로 핵심 기능 확인 불가.
9. DB 마이그레이션 실패 또는 롤백 불가.
10. 운영/CS/장애 대응 연락 체계 미준비.

## 13. 출시 완료 기준

출시는 다음 조건을 모두 만족할 때 완료로 본다.

- App Store와 Google Play 중 배포 대상 마켓에서 앱이 정상 검색/설치된다.
- 신규 사용자가 회원가입 또는 로그인 후 급여 계획을 등록할 수 있다.
- 급여 홈에서 납치금액과 일일 예산이 정상 계산된다.
- 운영 대시보드에서 가입, 로그인, 지출 기록, API 오류, 알림 발송 지표가 확인된다.
- CS 접수 채널이 동작한다.
- 장애 발생 시 롤백 계획서에 따라 의사결정 가능하다.
