# 화면 설계서 최종본

## 1. 문서 목적

본 문서는 급여납치 앱의 화면별 UI 요소, 입력값, 버튼, 상태값, 오류 메시지, 시스템 반응을 최종 확정한다. 개발자는 본 문서만으로 각 화면을 구현할 수 있어야 한다.

## 2. 공통 화면 요소

| 요소     | 적용 화면               | 기준                                |
| -------- | ----------------------- | ----------------------------------- |
| 상태바   | 전체 모바일 화면        | OS safe area 준수                   |
| 앱 헤더  | 로그인 후 화면          | 좌측 로고/앱명, 우측 알림           |
| 하단 탭  | 로그인 후 5개 주요 화면 | 급여, 계획, LV, 커뮤니티, MY        |
| 카드     | 정보 그룹               | 흰색 배경, 둥근 모서리, 얕은 그림자 |
| 기본 CTA | 저장/추가/완료          | 초록색 배경, 흰색 텍스트            |
| 보조 CTA | 설정/관리/취소          | 연한 초록 또는 흰색 배경            |
| 토스트   | 성공/간단 오류          | 화면 하단, 1.5~2초 표시             |
| 모달     | 위험/확인               | 중앙 또는 바텀시트                  |

## 3. SCR-001 스플래시

| 항목      | 명세                                         |
| --------- | -------------------------------------------- |
| 목적      | 앱 브랜드 인지 후 인증 상태에 따라 자동 이동 |
| UI 요소   | 로고, 앱명, 영문명, 운영사명                 |
| 표시 시간 | 1.2~2.0초                                    |
| 버튼      | 없음                                         |
| 상태      | 로딩, 네트워크 무관                          |
| 이동      | 로그인 상태면 SCR-005, 비로그인이면 SCR-002  |
| 오류      | 앱 초기화 실패 시 SCR-030 서버 오류          |

## 4. SCR-002 로그인

| 항목    | 명세                                                  |
| ------- | ----------------------------------------------------- |
| 목적    | 사용자 인증                                           |
| 입력값  | 아이디, 비밀번호                                      |
| 버튼    | 네이버, 카카오, 페이스북, 구글, 회원가입, 자동 로그인 |
| 검증    | 아이디/비밀번호 필수, 소셜 토큰 유효성                |
| 성공    | 급여 설정 여부 확인 후 SCR-004 또는 SCR-005 이동      |
| 실패    | “아이디 또는 비밀번호를 확인해주세요.” 표시           |
| 빈 상태 | 해당 없음                                             |

## 5. SCR-004 초기 급여 설정

| 항목   | 명세                                                              |
| ------ | ----------------------------------------------------------------- |
| 목적   | 첫 사용자의 급여관리 시작 데이터 수집                             |
| 입력값 | 급여일, 수령 예정 급여, 예상 고정지출, 예상 고정저축, 일일 생활비 |
| 버튼   | 다음, 저장하고 시작하기, 건너뛰기 불가                            |
| 검증   | 수령 예정 급여 1원 이상, 급여일 필수, 금액 숫자만 허용            |
| 성공   | PayrollPlan 생성 후 SCR-005 이동                                  |
| 실패   | 필드별 오류 문구 표시                                             |
| 계산   | 예상 납치금액 = 수령 예정 급여 - 예상 고정지출 - 예상 생활비 총액 |

## 6. SCR-005 급여 홈

| 항목        | 명세                                                |
| ----------- | --------------------------------------------------- |
| 목적        | 급여 상태와 오늘 쓸 수 있는 돈을 한 화면에서 확인   |
| 상단 정보   | 날짜, 이번 급여일, 다음 급여일                      |
| 핵심 수치   | 수령금액, 지출금액, 납치금액, 전체 누적 납치금액    |
| 카드        | 급여 현황, 광고 배너, 고정지출, 일일 예산, 변동지출 |
| 입력        | 빠른 지출명/금액 입력                               |
| 버튼        | 지출 추가, 사용완료, 카테고리 필터, 알림            |
| 상태        | 정상, 예산 초과, 지출 없음, 로딩, 오류              |
| 성공 메시지 | “지출이 추가되었습니다.”                            |
| 오류 메시지 | “금액을 숫자로 입력해주세요.”                       |

### 6.1 급여 홈 수치 표시 기준

| 수치          | 계산/표시 기준                            |
| ------------- | ----------------------------------------- |
| 수령금액      | 이번 급여 기간의 수령 또는 예정 급여      |
| 지출금액      | 고정지출 + 변동지출 + 생활비 사용액       |
| 납치금액      | 수령금액 - 지출금액, 음수일 경우 0원 표시 |
| 누적 납치금액 | 확정 월별 납치금액의 누적합               |
| 오늘 남은금액 | 일일 설정금액 - 오늘 사용금액             |

## 7. SCR-006 지출 추가/수정

| 항목   | 명세                               |
| ------ | ---------------------------------- |
| 목적   | 변동지출 생성/수정/삭제            |
| 입력값 | 카테고리, 항목명, 금액, 날짜, 메모 |
| 필수값 | 항목명, 금액, 날짜                 |
| 버튼   | 저장, 삭제, 취소                   |
| 검증   | 금액 1원 이상, 숫자만 허용         |
| 성공   | 홈 수치 재계산                     |
| 삭제   | 삭제 확인 모달 후 처리             |

## 8. SCR-007 알림

| 항목      | 명세                                                                      |
| --------- | ------------------------------------------------------------------------- |
| 목적      | 사용자 행동 유도와 상태 안내                                              |
| 알림 유형 | 목표 달성, 이벤트, 독서, 뉴스, 영어, 건강, 금융상식, 결제 예정, 예산 초과 |
| UI 요소   | 뒤로가기, 제목, 새 알림 표시, 알림 행                                     |
| 행 정보   | 아이콘, 카테고리, 제목, 요약, 시간, 읽음 상태                             |
| 클릭      | 관련 화면 이동 및 읽음 처리                                               |
| 빈 상태   | “새로운 알림이 없습니다.”                                                 |

## 9. SCR-008 계획

| 항목   | 명세                                                   |
| ------ | ------------------------------------------------------ |
| 목적   | 급여·지출·저축·생활비 계획 관리                        |
| 카드   | 목표 달성률, 급여 계획, 고정지출, 고정저축, 생활비     |
| 입력값 | 수령 예정 급여, 지출 예정 금액, 목표 누적 금액, 급여일 |
| 버튼   | 계획 저장 및 재계산, 항목 추가, 설정                   |
| 성공   | “급여 계획이 저장되었습니다.”                          |
| 오류   | 필수값 누락, 잘못된 금액, 저장 실패                    |

## 10. SCR-012 LV UP

| 항목      | 명세                                               |
| --------- | -------------------------------------------------- |
| 목적      | 자기계발 미션 수행과 레벨 성장 유도                |
| UI 요소   | 배너, 오늘의 미션, 현재 레벨, 경험치 바, 콘텐츠 탭 |
| 버튼      | 독서하기, 뉴스보기, 연습하기, 운동하기, 도전       |
| 완료 처리 | 미션 완료 API 호출, 경험치 증가                    |
| 레벨업    | 경험치 100% 이상 시 레벨 +1, 잔여 경험치 이월      |
| 오류      | 이미 완료한 미션이면 중복 완료 차단                |

## 11. SCR-013~016 레벨업 상세 화면

| 화면 | 핵심 UI                                       | 완료 기준                |
| ---- | --------------------------------------------- | ------------------------ |
| 독서 | 카테고리 탭, 도서 카드, 추천 사유, 도전 버튼  | 도전/완료 시 경험치 반영 |
| 뉴스 | 경제/산업/사회/기술 탭, 뉴스 카드             | 기사 읽음 후 완료 처리   |
| 영어 | Listening/Speaking/Reading/Writing, 일일 문장 | 학습 완료 처리           |
| 건강 | 요일별 홈트, 건강 지표, 완료 버튼             | 운동 완료 처리           |

## 12. SCR-017 커뮤니티

| 항목        | 명세                                                   |
| ----------- | ------------------------------------------------------ |
| 목적        | 직장인 자기관리 커뮤니티 제공                          |
| 탭          | 전체, 자유, 레벨업 인증, 취미                          |
| 카드        | 인기 게시글, 게시글 목록                               |
| 게시글 정보 | 썸네일, 제목, 요약, 게시판, 조회수, 좋아요, 댓글, 공유 |
| 버튼        | 게시판 탭, 글쓰기 FAB, 게시글 선택                     |
| 빈 상태     | “아직 게시글이 없습니다.”                              |

## 13. SCR-019 글쓰기

| 항목   | 명세                                           |
| ------ | ---------------------------------------------- |
| 목적   | 커뮤니티 게시글 작성                           |
| 입력값 | 게시판, 제목, 본문, 첨부, 질문 여부, 익명 여부 |
| 필수값 | 게시판, 제목, 본문                             |
| 버튼   | 닫기, 완료, 첨부                               |
| 검증   | 제목 2~80자, 본문 5~5000자, 첨부 용량 제한     |
| 성공   | 게시글 등록 후 목록 최상단 반영                |
| 오류   | “제목과 본문을 입력해주세요.”                  |

## 14. SCR-021 마이페이지

| 항목    | 명세                                                                     |
| ------- | ------------------------------------------------------------------------ |
| 목적    | 사용자 프로필, 성과, 활동 관리                                           |
| UI 요소 | 아바타, 닉네임, 직무, 프로필 설정, 계정 설정, 성과 카드, 관리 메뉴, 배너 |
| 성과    | 누적 납치금액, 레벨업 현황, 자기관리 성과                                |
| 메뉴    | 내 게시글 관리, 내 레벨업 관리, 1:1 문의, 공지사항                       |
| 오류    | 사용자 정보 로딩 실패 시 재시도 표시                                     |

## 15. 공통 오류 메시지

| 상황          | 메시지                                              |
| ------------- | --------------------------------------------------- |
| 필수값 누락   | 필수 항목을 입력해주세요.                           |
| 금액 오류     | 금액은 숫자로만 입력해주세요.                       |
| 저장 실패     | 저장하지 못했습니다. 다시 시도해주세요.             |
| 삭제 확인     | 삭제한 내용은 복구할 수 없습니다. 삭제하시겠습니까? |
| 네트워크 오류 | 연결이 불안정합니다. 네트워크 상태를 확인해주세요.  |
| 권한 없음     | 로그인이 필요한 기능입니다.                         |
| 글쓰기 오류   | 제목과 본문을 입력해주세요.                         |
| 파일 오류     | 첨부할 수 없는 파일 형식입니다.                     |
