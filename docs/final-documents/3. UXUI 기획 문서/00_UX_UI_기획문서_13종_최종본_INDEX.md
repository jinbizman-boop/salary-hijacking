## 1. 문서 세트 목적

본 문서 세트는 급여납치 플랫폼의 UX/UI 설계를 문서상·이론상 더 이상 추가 작업이 필요 없는 최종본으로 확정하기 위한 산출물이다. 화면 구조, 사용자 흐름, 와이어프레임, 화면 설계, 인터랙션, 프로토타입, 디자인 시스템, 컴포넌트, 반응형, 마이크로카피, 접근성까지 앱 구현에 필요한 UX/UI 기준을 모두 포함한다.

## 4. 적용 원칙

1. 본 문서 세트는 급여납치 모바일 앱 UX/UI 구현의 최종 기준이다.
2. 본 문서 세트와 기존 PRD/BRD/MVP/용어정의서가 충돌할 경우, 기능 범위는 PRD를 우선하고 화면 표현은 본 UX/UI 문서를 우선한다.
3. 문서에 명시되지 않은 신규 화면, 신규 인터랙션, 신규 디자인 토큰은 별도 변경 승인 없이는 추가하지 않는다.
4. 개발자는 화면 ID, 컴포넌트 ID, 상태 ID를 기준으로 작업 단위를 분해한다.
5. QA는 본 문서의 완료 기준과 체크리스트를 기준으로 검수한다.
