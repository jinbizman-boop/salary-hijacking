# 컴포넌트 명세서 최종본

## 1. 문서 목적

본 문서는 급여납치 앱의 재사용 UI 컴포넌트 구조, 속성, 상태, 이벤트, 데이터 바인딩 기준을 최종 정의한다. 프론트엔드 개발자는 본 문서를 기준으로 컴포넌트 단위 구현과 재사용을 수행한다.

## 2. 컴포넌트 ID 체계

| 구분 | 예시     | 설명                    |
| ---- | -------- | ----------------------- |
| CMP  | CMP-001  | 재사용 UI 컴포넌트      |
| PROP | PROP-001 | 컴포넌트 입력 속성      |
| EVT  | EVT-001  | 컴포넌트 이벤트         |
| SLOT | SLOT-001 | 컴포넌트 내부 삽입 영역 |

## 4. CMP-001 AppHeader

| 속성         | 타입        | 필수   | 설명                  |
| ------------ | ----------- | ------ | --------------------- |
| title        | string      | 예     | 표시 제목 또는 앱명   |
| showLogo     | boolean     | 아니오 | 로고 노출 여부        |
| rightAction  | icon/action | 아니오 | 알림 버튼             |
| onRightClick | function    | 아니오 | 우측 버튼 클릭 이벤트 |

| 상태      | 설명                |
| --------- | ------------------- |
| default   | 로고와 앱명 표시    |
| titleOnly | 상세 화면 제목 표시 |
| back      | 뒤로가기 포함       |

## 5. CMP-002 BottomNavigation

| 속성      | 타입     | 설명                                    |
| --------- | -------- | --------------------------------------- |
| activeTab | enum     | salary, plan, level, community, profile |
| items     | array    | 탭 목록                                 |
| onChange  | function | 탭 선택 이벤트                          |

| 탭        | 아이콘 | 라벨     | 라우트     |
| --------- | ------ | -------- | ---------- |
| salary    | 💸     | 급여     | /home      |
| plan      | 💼     | 계획     | /plan      |
| level     | ⬆️     | LV       | /level     |
| community | 💬     | 커뮤니티 | /community |
| profile   | 👤     | MY       | /profile   |

## 6. CMP-003 SalaryHeroCard

| 속성              | 타입        | 설명          |
| ----------------- | ----------- | ------------- |

| expenseAmount     | number      | 지출금액      |

## 7. CMP-006 DailyBudgetCard

| 속성            | 타입      | 설명          |
| --------------- | --------- | ------------- |
| dailyLimit      | number    | 일일 설정금액 |
| usedAmount      | number    | 사용금액      |
| remainingAmount | number    | 남은금액      |
| expenses        | Expense[] | 변동지출 목록 |
| onAddExpense    | function  | 지출 추가     |

| 상태     | 표시 기준                        |
| -------- | -------------------------------- |
| normal   | remainingAmount >= 0             |
| exceeded | remainingAmount < 0, 빨간색 표시 |
| empty    | expenses.length === 0            |
