# 와이어프레임 문서 최종본

## 1. 문서 목적

본 문서는 급여납치 앱의 주요 화면 레이아웃, 카드 배치, 버튼 위치, 탭 구조, 정보 우선순위를 디자인 이전 단계에서 최종 확정한다. 모든 시각 디자인은 본 와이어프레임 구조를 유지해야 한다.

## 2. 공통 모바일 프레임

```text
┌─────────────────────────┐
│ Status Bar              │
├─────────────────────────┤
│ App Header / Title / 🔔 │
├─────────────────────────┤
│                         │
│ Scrollable Content      │
│                         │
├─────────────────────────┤
│ 💸 급여 │ 💼 계획 │ ⬆ LV │ 💬 커뮤니티 │ 👤 MY │
└─────────────────────────┘
```

## 3. 공통 레이아웃 규칙

| 영역        | 규칙                                   |
| ----------- | -------------------------------------- |
| 상태바      | OS 기본 safe area를 침범하지 않는다.   |
| 앱 헤더     | 로그인 이후 모든 주요 탭에서 노출한다. |
| 콘텐츠      | 세로 스크롤을 기본으로 한다.           |
| 하단 탭     | 로그인 이후 항상 고정한다.             |
| 플로팅 버튼 | 커뮤니티 글쓰기에서만 사용한다.        |
| 광고 배너   | 핵심 급여 수치보다 우선하지 않는다.    |
| 카드        | 정보 그룹 단위로 흰색 카드에 담는다.   |

## 13. 와이어프레임 불변 규칙

1. 하단 탭은 모든 로그인 후 주요 화면에서 동일 위치에 유지한다.
2. 급여 홈의 초록 히어로 카드는 화면 최상단 핵심 영역으로 유지한다.
3. 일일 예산 카드에는 설정금액, 사용금액, 남은금액을 같은 행에 배치한다.
4. 계획 화면은 목표 달성률 → 급여 계획 → 고정지출 → 고정저축 → 생활비 순서로 배치한다.
5. LV UP은 오늘 수행할 행동 버튼을 콘텐츠 목록보다 먼저 배치한다.
6. 커뮤니티 글쓰기 FAB는 커뮤니티 화면에서만 노출한다.
7. 마이페이지는 성과 카드와 관리 메뉴를 분리한다.
8. 광고 배너는 핵심 기능 버튼과 혼동되지 않게 시각적으로 구분한다.
