# 마이크로카피 문서 최종본

## 1. 문서 목적

본 문서는 급여납치 앱에서 사용하는 오류 문구, 성공 문구, 알림 문구, 빈 상태 문구, 온보딩 문구, 버튼 문구를 최종 확정한다. 모든 화면의 사용자 안내 문구는 본 문서를 기준으로 통일한다.

## 2. 카피 톤 원칙

| 원칙      | 기준                                                 |
| --------- | ---------------------------------------------------- |
| 직접적    | 사용자가 무엇을 해야 하는지 명확히 말한다.           |
| 긍정적    | “못 쓰게 막음”보다 “지켜냄”을 강조한다.              |
| 과장 금지 | 투자 수익, 절약 효과, 건강 효과를 보장하지 않는다.   |
| 짧고 명확 | 모바일 화면에서 한 번에 이해 가능한 길이로 작성한다. |
| 신뢰 우선 | 급여·지출 데이터는 민감하므로 안심 문구를 제공한다.  |

## 3. 브랜드 핵심 문구

| 위치          | 문구                                                                   |
| ------------- | ---------------------------------------------------------------------- |
| 메인 슬로건   | 월급이 사라지기 전에, 내가 먼저 붙잡아두는 급여 자기관리 앱            |
| 서비스 설명   | 이번 월급에서 얼마를 썼는지보다, 얼마를 지켜냈는지 보여드립니다.       |
| 납치금액 설명 | 납치금액은 이번 급여에서 지켜낸 금액입니다.                            |
| 온보딩 문구   | 급여일과 생활비를 설정하면 오늘 쓸 수 있는 금액을 바로 계산해드립니다. |
| 신뢰 문구     | 급여와 지출 정보는 사용자 본인만 확인할 수 있도록 보호됩니다.          |

## 6. 오류 문구

| 상황             | 문구                                                        |
| ---------------- | ----------------------------------------------------------- |
| 필수값 누락      | 필수 항목을 입력해주세요.                                   |
| 로그인 실패      | 아이디 또는 비밀번호를 확인해주세요.                        |
| 소셜 로그인 실패 | 소셜 로그인에 실패했습니다. 다시 시도해주세요.              |
| 금액 오류        | 금액은 숫자로만 입력해주세요.                               |
| 음수 입력        | 0원 미만의 금액은 입력할 수 없습니다.                       |
| 급여보다 지출 큼 | 지출 예정 금액이 급여보다 큽니다. 계획을 다시 확인해주세요. |
| 제목 누락        | 제목을 입력해주세요.                                        |
| 본문 누락        | 본문을 입력해주세요.                                        |
| 첨부 오류        | 첨부할 수 없는 파일 형식입니다.                             |
| 네트워크 오류    | 연결이 불안정합니다. 네트워크 상태를 확인해주세요.          |
| 서버 오류        | 잠시 후 다시 시도해주세요.                                  |
| 권한 없음        | 로그인이 필요한 기능입니다.                                 |
| 중복 미션        | 오늘 이미 완료한 미션입니다.                                |
| 게시글 삭제됨    | 삭제되었거나 볼 수 없는 게시글입니다.                       |
