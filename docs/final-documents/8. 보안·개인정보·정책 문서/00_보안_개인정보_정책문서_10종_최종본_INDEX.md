# 00. 보안·개인정보·정책 문서 10종 최종본 INDEX

## 1. 문서 세트 개요

본 폴더는 급여납치 플랫폼의 보안·개인정보·정책 구현 및 출시를 위한 최종본 Markdown 문서 10종으로 구성된다. 급여납치 플랫폼은 급여, 대출, 저축, 소비 내역, 생활비 계획, 자기계발 활동, 커뮤니티 게시글, 광고/제휴 반응 데이터를 처리하므로 일반 가계부보다 높은 개인정보 보호 기준을 적용한다.

본 세트는 다음 원칙을 최종 확정한다.

1. 급여·대출·저축·소비 내역은 법정 민감정보와 별도로 서비스 내부에서 `고위험 재무성 개인정보`로 상향 분류한다.
2. 광고/제휴 데이터는 급여·대출·저축·소비 원천 데이터와 분리한다.
3. 앱 권한은 기능 실행 시점에 최소한으로 요청한다.
4. 커뮤니티 익명성은 화면상 보장하되, 신고·불법행위 대응을 위한 내부 식별자는 암호화/접근통제 하에 보관한다.
5. 뉴스·건강·금융성 콘텐츠는 정보 제공 목적이며 투자·의료·법률·세무 자문으로 표시하지 않는다.
6. 개인정보처리방침, 약관, 동의 화면, 권한 안내, 보안 요구사항은 출시 문서와 운영 정책에 직접 연결된다.

## 4. 최종 완료 선언

본 보안·개인정보·정책 문서 10종은 급여납치 플랫폼의 개인정보 처리, 민감도 분류, 동의 UI, 모바일 권한, 보안 요구사항, 광고 데이터 분리, 커뮤니티 운영, 콘텐츠 면책, 약관, 개인정보처리방침에 필요한 모든 기준을 포함한다. 따라서 본 세트는 문서상·이론상 더 이상 수행할 작업이 없는 최종본으로 확정한다.
