# 07. 커뮤니티 운영 정책서 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼의 커뮤니티 게시판 운영 기준을 정의한다. 급여납치 커뮤니티는 직장인·아르바이트생의 급여관리, 자기계발 인증, 자유 소통, 취미 공유를 목적으로 하며, 익명 기능을 제공하되 불법행위와 오정보 확산을 방지하기 위해 신고·숨김·차단·제재 정책을 적용한다.

## 2. 커뮤니티 게시판 구조

| 게시판      | 목적                  | 주요 콘텐츠                      | 익명 허용 |   운영 강도 |
| ----------- | --------------------- | -------------------------------- | --------: | ----------: |
| 전체 게시판 | 모든 게시글 통합 노출 | 인기글, 최신글                   |        예 |        보통 |
| 자유 게시판 | 직장생활/일상/유머    | 자유 글, 질문                    |        예 |        보통 |
| 레벨업 인증 | 자기계발 인증         | 독서, 뉴스, 영어, 운동 완료 인증 |        예 |        높음 |
| 취미 게시판 | 취미/부업/루틴 공유   | 취미, 생산성, 생활 팁            |        예 |        보통 |
| 공지사항    | 운영 공지             | 업데이트, 점검, 정책             |    아니오 | 관리자 전용 |

## 3. 기본 운영 원칙

| 원칙            | 기준                                                                         |
| --------------- | ---------------------------------------------------------------------------- |
| 익명성 보장     | 화면상 작성자 정보는 닉네임 또는 익명으로 표시한다.                          |
| 내부 식별 보관  | 신고/분쟁/불법행위 대응을 위해 내부 user_id는 암호화·접근통제 하에 보관한다. |
| 오정보 억제     | 투자, 금융, 건강, 법률, 세무성 글은 면책 문구와 신고 기준을 적용한다.        |
| 차별/혐오 금지  | 성별, 지역, 직업, 고용형태, 장애, 국적, 종교 등에 대한 혐오 표현을 금지한다. |
| 상업행위 제한   | 허가 없는 광고, 리워드 링크, 사기성 부업 홍보를 금지한다.                    |
| 안전한 커뮤니티 | 자해, 폭력, 불법행위, 개인정보 유출, 스토킹을 금지한다.                      |

## 7. 제재 단계

|  단계 | 조치             | 조건                                                  |
| ----: | ---------------- | ----------------------------------------------------- |
| 1단계 | 안내/수정 요청   | 경미한 문구 위반                                      |
| 2단계 | 게시글 숨김/삭제 | 정책 위반 확인                                        |
| 3단계 | 24시간 작성 제한 | 반복 위반 또는 신고 누적                              |
| 4단계 | 7일 이용 제한    | 심각한 괴롭힘, 광고, 사기성 글                        |
| 5단계 | 30일 이용 제한   | 반복 중대 위반                                        |
| 6단계 | 영구 정지        | 불법행위, 개인정보 대량 유출, 사기, 음란물, 폭력 위협 |

## 11. 최종 완료 선언

본 커뮤니티 운영 정책서는 게시판 구조, 금지 콘텐츠, 신고, 자동 숨김, 제재, 익명글, 차단, 운영자 처리 기록 기준을 모두 포함한다. 따라서 본 문서는 문서상·이론상 더 이상 수행할 작업이 없는 최종본으로 확정한다.
