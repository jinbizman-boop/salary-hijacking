# 10. 개인정보처리방침 초안 최종본

> 본 문서는 급여납치 플랫폼의 외부 고지용 개인정보처리방침 초안 최종본이다. 실제 앱 화면에는 회사명, 대표자, 주소, 사업자등록번호, 개인정보 보호책임자, 고객센터 이메일 등 운영 주체 정보를 확정값으로 삽입한다. 문서상·이론상 개인정보처리방침에 필요한 항목은 본 문서에 모두 포함한다.

# 급여납치 개인정보처리방침

급여납치(Salary Hijacking)는 이용자의 개인정보를 중요하게 생각하며, 개인정보 보호 관련 법령을 준수하고 이용자의 개인정보를 안전하게 처리하기 위해 다음과 같이 개인정보처리방침을 수립·공개합니다.

## 1. 개인정보의 처리 목적

급여납치는 다음 목적을 위해 개인정보를 처리합니다.

| 목적                  | 상세 내용                                                                |
| --------------------- | ------------------------------------------------------------------------ |
| 회원가입 및 계정관리  | 회원 식별, 로그인, 소셜 계정 연동, 중복 가입 방지, 계정 보호             |
| 급여관리 서비스 제공  | 급여일, 급여계획, 고정지출, 고정저축, 일일 예산, 변동지출, 납치금액 계산 |
| 자기계발 레벨업 제공  | 독서, 뉴스, 영어, 건강 미션, 경험치, 레벨, 활동 기록 표시                |
| 커뮤니티 제공         | 게시글, 댓글, 좋아요, 조회수, 신고, 익명 글쓰기, 운영정책 집행           |
| 알림 제공             | 급여일, 예산 초과, 목표 달성, 결제 예정, 레벨업, 이벤트 알림             |
| 고객지원              | 문의 접수, 오류 확인, 신고 처리, 분쟁 대응                               |
| 보안 및 부정이용 방지 | 접속기록, 인증 실패, 비정상 활동 탐지, 계정 보호                         |
| 광고/제휴 운영        | 광고 노출, 클릭, 정산, 비개인화 광고, 선택 동의 기반 맞춤형 광고         |
| 서비스 개선           | 기능 사용 분석, 오류 개선, 품질 향상, 통계 작성                          |

## 2. 처리하는 개인정보 항목

### 2.1 회원가입 및 계정 정보

| 구분      | 항목                                                                      |
| --------- | ------------------------------------------------------------------------- |
| 필수      | 이메일, 비밀번호 해시, 닉네임, 만 14세 이상 여부, 약관/개인정보 동의 내역 |
| 선택      | 프로필 이미지, 직무/직업군, 소셜 로그인 제공자 정보                       |
| 자동 생성 | user_id, 가입일, 최근 로그인일, 기기 세션 정보                            |

### 2.2 급여·예산·지출 정보

| 구분      | 항목                                                   |
| --------- | ------------------------------------------------------ |
| 필수/입력 | 급여일, 수령 예정 급여, 지출 예정 금액, 일일 예산      |
| 선택/입력 | 고정지출 항목, 고정저축 항목, 변동지출 항목, 소비 메모 |
| 자동 생성 | 예상 납치금액, 누적 납치금액, 목표 달성률, 남은금액    |

급여·대출·저축·소비 내역은 법정 민감정보와 별도로 서비스 내부에서 고위험 재무성 개인정보로 분류하여 보호합니다.

### 2.3 커뮤니티 및 콘텐츠 이용 정보

| 구분      | 항목                                                            |
| --------- | --------------------------------------------------------------- |

| 레벨업    | 미션 완료 이력, 레벨, 경험치, 콘텐츠 조회 이력                  |
| 알림      | 푸시 토큰, 알림 수신 설정, 알림 읽음 여부                       |

## 4. 개인정보의 제3자 제공

급여납치는 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 다만 다음의 경우에는 예외적으로 제공할 수 있습니다.

1. 이용자가 사전에 명시적으로 동의한 경우
2. 법령에 근거가 있거나 수사기관 등 적법한 절차에 따른 요청이 있는 경우
3. 생명·신체·재산의 급박한 위험을 방지할 필요가 있는 경우

급여액, 대출액, 저축액, 소비 원천 내역은 광고주에게 제공하지 않습니다.

## 5. 개인정보 처리의 위탁

서비스 운영을 위해 다음 업무를 외부 서비스에 위탁할 수 있습니다. 실제 위탁사는 운영 환경 확정 시 앱 내 개인정보처리방침에 명시합니다.

| 위탁 업무       | 수탁자 예시                      | 처리 목적                      |
| --------------- | -------------------------------- | ------------------------------ |
| 클라우드 인프라 | Cloudflare                       | API 보호, CDN, 파일 전달, 보안 |
| 데이터베이스    | Neon DB                          | 개인정보 및 서비스 데이터 저장 |
| 푸시 알림       | Firebase Cloud Messaging         | 앱 푸시 발송                   |
| 소셜 로그인     | 네이버, 카카오, Google, Apple 등 | 로그인 인증                    |
| 오류 분석       | 로그/모니터링 도구               | 장애 및 오류 분석              |

위탁사는 개인정보 보호를 위한 계약 및 접근통제 기준에 따라 관리합니다.

## 6. 개인정보의 파기

급여납치는 보유기간 경과, 처리 목적 달성, 회원 탈퇴, 이용자 삭제 요청이 있는 경우 지체 없이 개인정보를 파기합니다.

| 유형            | 파기 방식                                       |
| --------------- | ----------------------------------------------- |
| DB 데이터       | 레코드 삭제 또는 익명화                         |
| 파일            | 저장소에서 삭제 및 CDN 캐시 무효화              |
| 백업            | 백업 보존기간 경과 후 순차 삭제                 |
| 로그            | 보존기간 경과 후 삭제 또는 집계화               |
| 커뮤니티 데이터 | 사용자 선택 및 운영정책에 따라 삭제 또는 익명화 |

## 7. 이용자의 권리와 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

1. 개인정보 열람 요청
2. 오류 정정 요청
3. 삭제 요청
4. 처리정지 요청
5. 동의 철회
6. 광고/마케팅 수신 거부
7. 회원 탈퇴

권리 행사는 앱 내 `마이페이지 > 설정 > 개인정보 관리` 또는 고객센터를 통해 할 수 있습니다.

## 8. 만 14세 미만 아동의 개인정보 처리

급여납치는 만 14세 미만 이용자의 가입을 허용하지 않습니다. 만 14세 미만 이용자의 가입 또는 개인정보 처리가 확인되는 경우 해당 계정을 제한하고 개인정보를 삭제합니다.

## 9. 개인정보의 안전성 확보조치

급여납치는 개인정보 보호를 위해 다음 조치를 적용합니다.

| 영역           | 조치                                                           |
| -------------- | -------------------------------------------------------------- |
| 접근통제       | 최소 권한, 관리자 권한 분리, MFA, 감사로그                     |
| 암호화         | 전송구간 암호화, 비밀번호 일방향 해시, 주요 재무성 필드 암호화 |
| 접속기록       | 로그인, 관리자 조치, 보안 이벤트 기록                          |
| 악성 행위 방지 | rate limit, 입력값 검증, 파일 업로드 제한                      |
| 로그 보호      | 토큰, 비밀번호, 재무 원천 데이터 로그 저장 금지                |
| 백업 보호      | 암호화 백업 및 보존기간 후 삭제                                |
| 교육/관리      | 관리자 접근 권한 정기 점검 및 보안 정책 준수                   |

## 10. 광고 및 개인화 처리

1. 급여납치는 기본적으로 비개인화 또는 문맥형 광고를 제공합니다.
2. 맞춤형 광고는 이용자가 선택 동의한 경우에만 제공합니다.
3. 급여액, 대출액, 저축액, 소비 원천 내역은 광고 타겟팅에 직접 사용하지 않습니다.
4. 이용자는 설정에서 맞춤형 광고 동의를 언제든지 철회할 수 있습니다.

## 11. 앱 권한 안내

| --------- | ------------------------------------------------- | --------- |

| 사진/파일 | 커뮤니티 글쓰기, 프로필 이미지, 첨부파일          | 선택      |
| 카메라    | 직접 촬영 첨부                                    | 선택      |
| 생체 인증 | 앱 잠금 및 재무 화면 보호                         | 선택      |

선택 권한에 동의하지 않아도 기본 급여관리 기능은 이용할 수 있습니다.
