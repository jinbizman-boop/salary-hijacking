# 08. 콘텐츠 면책/안내 문서 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼이 제공하는 뉴스, 독서, 영어, 건강, 금융상식, 자기계발 콘텐츠의 안내 문구, 출처 표시, 면책 기준, 위험 고지 기준을 정의한다. 급여납치의 콘텐츠는 사용자 성장과 정보 습득을 돕기 위한 참고 정보이며, 전문 자문을 대체하지 않는다.

## 2. 콘텐츠 분류

| 콘텐츠 유형            | 예시                             | 위험도 | 주요 고지                             |
| ---------------------- | -------------------------------- | -----: | ------------------------------------- |
| 독서 추천              | 기획/경제/인문 도서 추천         |   낮음 | 추천 기준 및 광고/제휴 여부 표시      |
| 뉴스 요약              | 경제, 산업, 사회, 기술 뉴스      |   중간 | 출처, 발행일, 요약 기준 표시          |
| 영어 학습              | 문장, 듣기, 말하기, 읽기, 쓰기   |   낮음 | 학습 참고 자료 안내                   |
| 건강/홈트              | 운동 루틴, 회복, 영양 일반 정보  |   중간 | 의학적 조언 아님, 통증 시 중단 안내   |
| 금융상식               | 예산, 저축, 소비 습관, 금융 용어 |   중간 | 투자·대출 권유 아님                   |
| 제휴 콘텐츠            | 광고주 제공 배너/혜택            |   중간 | 광고/제휴 표시                        |
| 커뮤니티 사용자 콘텐츠 | 게시글, 댓글, 인증글             |   높음 | 사용자 의견이며 플랫폼 공식 의견 아님 |

## 3. 공통 면책 문구

| 위치             | 문구                                                                                                                                                  |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| 콘텐츠 상세 하단 | `본 콘텐츠는 정보 제공 및 자기계발 참고용입니다. 투자, 금융상품 가입, 의료, 법률, 세무 등 전문적인 판단이 필요한 사항은 관련 전문가와 상담해 주세요.` |
| 뉴스 요약 하단   | `뉴스 콘텐츠는 외부 출처를 바탕으로 요약·분류될 수 있으며, 원문과 시점 차이가 있을 수 있습니다.`                                                      |
| 건강 콘텐츠 하단 | `운동 중 통증, 어지러움, 호흡곤란 등 이상 증상이 있으면 즉시 중단하고 전문가와 상담해 주세요.`                                                        |
| 금융상식 하단    | `금융상식 콘텐츠는 일반 정보 제공 목적이며 특정 금융상품, 투자, 대출을 권유하지 않습니다.`                                                            |
| 커뮤니티 글 하단 | `커뮤니티 게시글은 작성자의 의견이며 급여납치의 공식 입장이 아닙니다.`                                                                                |
| 광고/제휴 콘텐츠 | `본 영역은 광고/제휴 콘텐츠입니다.`                                                                                                                   |

## 4. 뉴스 콘텐츠 기준

| 항목           | 기준                                               |
| -------------- | -------------------------------------------------- |
| 출처 표시      | 언론사/제공처/발행일/링크 표시                     |
| 요약 표시      | `요약` 또는 `AI 요약` 여부 표시                    |
| 원문 연결      | 가능하면 원문 보기 링크 제공                       |
| 오래된 정보    | 발행일이 30일 이상 지난 경우 날짜 강조             |
| 금융/투자 뉴스 | 특정 종목 매수·매도 권유 문구 금지                 |
| 저작권         | 기사 전문 무단 복제 금지, 제목/요약/링크 중심 제공 |

## 5. 건강/홈트 콘텐츠 기준

| 항목          | 기준                                                  |
| ------------- | ----------------------------------------------------- |
| 금지 표현     | 치료 보장, 질병 완치, 단기간 극단 감량 보장           |
| 허용 표현     | 일반적인 운동 루틴, 스트레칭, 휴식, 수분 섭취 안내    |
| 위험 고지     | 통증/질환/임신/부상 이력이 있는 경우 전문가 상담 안내 |
| 개인화 제한   | 건강 상태, 질병 정보 수집 없이 일반 미션 제공         |
| 커뮤니티 인증 | 과도한 신체 노출, 위험 운동, 극단 식단 조장 금지      |

## 6. 금융상식/급여관리 콘텐츠 기준

| 항목        | 기준                                                               |
| ----------- | ------------------------------------------------------------------ |
| 허용        | 예산 관리법, 소비 기록법, 고정지출 점검, 저축 습관, 금융 용어 설명 |
| 금지        | 특정 주식/코인/대출/보험/카드 가입 권유를 콘텐츠로 위장            |
| 제휴        | 제휴 배너는 광고로 명확히 표시                                     |
| 개인 데이터 | 사용자의 급여액, 대출액, 저축액을 콘텐츠 문구에 직접 삽입하지 않음 |
| 면책        | 투자·대출·세무 자문 아님을 명확히 표시                             |

## 7. 독서/영어 콘텐츠 기준

| 항목      | 기준                                          |
| --------- | --------------------------------------------- |
| 도서 추천 | 카테고리, 추천 사유, 제휴 여부 표시           |
| 저작권    | 도서 본문 장문 제공 금지, 짧은 소개/요약 중심 |
| 영어 문장 | 학습 목적 예문 제공, 출처 필요 시 표시        |
| 학습 기록 | 완료 여부, 경험치, 레벨업에만 활용            |

## 8. 사용자 생성 콘텐츠 안내

| 위치        | 문구                                                                                                           |
| ----------- | -------------------------------------------------------------------------------------------------------------- |
| 글쓰기 화면 | `타인의 개인정보, 회사 내부자료, 불법 투자/부업 홍보, 혐오·비방·음란·저작권 침해 콘텐츠는 게시할 수 없습니다.` |
| 게시글 상세 | `작성자의 의견이며 급여납치의 공식 검증을 받은 정보가 아닙니다.`                                               |
| 레벨업 인증 | `인증 사진에 타인의 얼굴, 개인정보, 회사 내부자료가 포함되지 않도록 확인해 주세요.`                            |
| 신고 버튼   | `부적절한 정보나 위험한 조언을 발견하면 신고해 주세요.`                                                        |

## 9. 광고/제휴 콘텐츠 표시 기준

| 유형          | 표시 문구                                                       |
| ------------- | --------------------------------------------------------------- |
| 배너 광고     | `광고`                                                          |
| 제휴 혜택     | `제휴` 또는 `제휴 혜택`                                         |
| 보상형 이벤트 | `이벤트 참여 시 포인트가 지급될 수 있습니다.`                   |
| 외부 링크     | `외부 사이트로 이동합니다. 해당 사이트의 정책을 확인해 주세요.` |
| 콘텐츠형 광고 | `본 콘텐츠는 제휴/광고를 포함합니다.`                           |

## 10. 콘텐츠 오류 정정 절차

| 단계           | 처리                                  |
| -------------- | ------------------------------------- |
| 오류 신고 접수 | 사용자 신고 또는 운영자 발견          |
| 위험도 분류    | 단순 오탈자, 정보 오류, 고위험 오정보 |
| 임시 조치      | 고위험 오정보는 즉시 숨김             |
| 검토           | 출처 확인 및 수정                     |
| 재게시         | 수정일 표시                           |
| 이력 보관      | 수정 전후 이력 운영 로그 보관         |

## 11. 최종 완료 선언

본 콘텐츠 면책/안내 문서는 뉴스, 건강, 금융상식, 독서, 영어, 커뮤니티, 광고/제휴 콘텐츠의 위험 고지, 출처 표시, 면책 문구, 오류 정정 기준을 모두 포함한다. 따라서 본 문서는 문서상·이론상 더 이상 수행할 작업이 없는 최종본으로 확정한다.
