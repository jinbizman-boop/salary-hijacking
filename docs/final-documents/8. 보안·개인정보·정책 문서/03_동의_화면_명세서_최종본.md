# 03. 동의 화면 명세서 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼의 회원가입, 소셜 로그인, 개인정보 수집·이용, 마케팅 수신, 광고 개인화, 푸시 알림, 커뮤니티 운영에 필요한 동의 화면의 구조, 문구, 필수/선택 구분, 저장 방식, 철회 경로를 정의한다.

## 2. 동의 설계 원칙

| 원칙               | 확정 기준                                                                                               |
| ------------------ | ------------------------------------------------------------------------------------------------------- |
| 분리 동의          | 서비스 이용 필수 동의와 선택 동의를 분리한다.                                                           |
| 명확한 문구        | 동의 대상, 수집 항목, 이용 목적, 보유기간을 한 화면에서 확인 가능하게 한다.                             |
| 미동의 불이익 금지 | 선택 동의 거부 시 핵심 서비스 이용을 제한하지 않는다.                                                   |
| 철회 가능성        | 마이페이지 > 설정에서 모든 선택 동의를 철회할 수 있다.                                                  |
| 이력 저장          | 동의 버전, 시각, IP, 기기, 동의 여부를 consent_logs에 저장한다.                                         |
| 만 14세 미만 제한  | 급여납치 기본 정책은 만 14세 미만 가입 불가이며, 추후 허용 시 법정대리인 동의 플로우를 별도 활성화한다. |

## 3. 회원가입 동의 화면 구조

```text
회원가입/소셜 로그인 완료 전
  ├─ 전체 동의 체크박스
  ├─ [필수] 서비스 이용약관 동의
  ├─ [필수] 개인정보 수집·이용 동의
  ├─ [필수] 만 14세 이상 확인
  ├─ [선택] 마케팅 정보 수신 동의
  ├─ [선택] 맞춤형 콘텐츠/광고 개인화 동의
  └─ 동의하고 시작하기 버튼
```

## 4. 동의 항목 정의

| 동의 ID                         | 항목명                     | 필수 여부   | 연결 문서                    | 동의하지 않을 경우       | 철회 경로               |
| ------------------------------- | -------------------------- | ----------- | ---------------------------- | ------------------------ | ----------------------- |
| CONSENT_TERMS_V1                | 서비스 이용약관 동의       | 필수        | 약관 초안                    | 가입 불가                | 탈퇴로 종료             |
| CONSENT_PRIVACY_V1              | 개인정보 수집·이용 동의    | 필수        | 개인정보처리방침             | 가입 불가                | 탈퇴로 종료             |
| CONSENT_AGE14_V1                | 만 14세 이상 확인          | 필수        | 개인정보처리방침             | 가입 불가                | 해당 없음               |
| CONSENT_PUSH_SERVICE_V1         | 서비스 필수 알림 수신 안내 | 필수 안내   | 권한 정책서                  | 앱 내 알림만 제공 가능   | 알림 설정에서 일부 조정 |
| CONSENT_MARKETING_PUSH_V1       | 마케팅/이벤트 알림 수신    | 선택        | 동의 화면 명세서             | 이벤트/광고 알림 미수신  | 설정 > 알림             |
| CONSENT_PERSONALIZED_CONTENT_V1 | 맞춤형 콘텐츠 추천         | 선택        | 콘텐츠 면책/개인정보처리방침 | 일반 콘텐츠만 노출       | 설정 > 개인정보         |
| CONSENT_PERSONALIZED_ADS_V1     | 맞춤형 광고/제휴 추천      | 선택        | 광고 데이터 분리 정책서      | 비개인화 광고만 노출     | 설정 > 광고 개인정보    |
| CONSENT_THIRD_PARTY_V1          | 제3자 제공 동의            | 선택/상황별 | 개인정보처리방침             | 해당 제휴 기능 이용 불가 | 설정 > 제휴 동의        |

## 5. 필수 동의 문구

### 5.1 서비스 이용약관 동의

| 항목        | 화면 문구                                                                                     |
| ----------- | --------------------------------------------------------------------------------------------- |
| 체크박스    | `[필수] 급여납치 서비스 이용약관에 동의합니다.`                                               |
| 상세 설명   | `서비스 이용 조건, 계정 관리, 커뮤니티 이용, 금지행위, 책임 범위에 대한 내용을 확인했습니다.` |
| 링크        | `약관 보기`                                                                                   |
| 미동의 처리 | `필수 약관에 동의해야 급여납치 서비스를 이용할 수 있습니다.`                                  |

### 5.2 개인정보 수집·이용 동의

| 항목           | 화면 문구                                                                                               |
| -------------- | ------------------------------------------------------------------------------------------------------- |
| 체크박스       | `[필수] 개인정보 수집·이용에 동의합니다.`                                                               |
| 상세 설명      | `계정 생성, 급여 계획 관리, 지출 기록, 알림, 고객지원 제공을 위해 필요한 개인정보를 처리합니다.`        |
| 주요 수집 항목 | `이메일, 닉네임, 로그인 정보, 급여일, 급여 계획, 지출/저축 계획, 알림 설정, 서비스 이용기록`            |
| 보유기간       | `회원 탈퇴 시까지. 단, 관계 법령 또는 운영 정책상 보존이 필요한 정보는 정해진 기간 보관 후 삭제합니다.` |
| 미동의 처리    | `개인정보 수집·이용에 동의해야 서비스 이용이 가능합니다.`                                               |

### 5.3 만 14세 이상 확인

| 항목        | 화면 문구                                                                                                |
| ----------- | -------------------------------------------------------------------------------------------------------- |
| 체크박스    | `[필수] 만 14세 이상입니다.`                                                                             |
| 상세 설명   | `급여납치 서비스는 급여·소비·커뮤니티 정보를 처리하므로 만 14세 미만 사용자의 가입을 허용하지 않습니다.` |
| 미동의 처리 | `만 14세 이상 확인이 필요합니다.`                                                                        |

## 6. 선택 동의 문구

### 6.1 마케팅/이벤트 알림 수신 동의

| 항목        | 화면 문구                                                                         |
| ----------- | --------------------------------------------------------------------------------- |
| 체크박스    | `[선택] 이벤트·혜택·제휴 소식 알림을 받겠습니다.`                                 |
| 설명        | `포인트 이벤트, 제휴 혜택, 광고성 알림을 푸시 또는 앱 알림으로 받을 수 있습니다.` |
| 수신 채널   | `앱 푸시, 앱 내 알림`                                                             |
| 철회        | `설정 > 알림 설정에서 언제든지 끌 수 있습니다.`                                   |
| 미동의 영향 | `동의하지 않아도 급여관리, 지출기록, 레벨업, 커뮤니티 기능은 이용할 수 있습니다.` |

### 6.2 맞춤형 콘텐츠 추천 동의

| 항목      | 화면 문구                                                               |
| --------- | ----------------------------------------------------------------------- |

| 설명      | `독서, 뉴스, 영어, 건강 미션 완료 이력을 바탕으로 콘텐츠를 추천합니다.` |
| 제외 기준 | `급여, 대출, 저축 원천 금액은 콘텐츠 추천에 직접 사용하지 않습니다.`    |
| 철회      | `설정 > 개인정보 설정에서 철회할 수 있습니다.`                          |

### 6.3 맞춤형 광고/제휴 추천 동의

| 금지 문구 | `급여액, 대출액, 저축액, 지출 원천 내역은 광고 타겟팅에 직접 사용하지 않습니다.`     |
| 철회      | `설정 > 광고 개인정보 설정에서 철회할 수 있습니다.`                                  |

## 10. 최종 완료 선언

본 동의 화면 명세서는 필수/선택 동의, 광고 수신 동의, 개인정보 동의, 제3자 제공, 권한 고지, 철회 경로, 동의 로그 저장 기준을 모두 포함한다. 따라서 본 문서는 문서상·이론상 더 이상 수행할 작업이 없는 최종본으로 확정한다.
