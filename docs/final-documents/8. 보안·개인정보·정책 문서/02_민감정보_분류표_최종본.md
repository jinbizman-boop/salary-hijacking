# 02. 민감정보 분류표 최종본

## 1. 문서 목적

본 문서는 급여납치 플랫폼에서 처리하는 모든 데이터의 위험도, 보호 등급, 접근 권한, 암호화 여부, 로그 저장 허용 여부를 정의한다. 급여·대출·저축·소비 내역은 법정 민감정보와 별도로 서비스 내부에서 `고위험 재무성 개인정보`로 상향 분류한다.
