# 04. 권한 정책서 최종본

## 1. 문서 목적

본 문서는 급여납치 모바일 앱이 요청하는 Android/iOS 권한의 목적, 요청 시점, 사용자 안내 문구, 거부 시 대체 흐름, 재요청 기준을 정의한다. 급여납치 앱은 권한 최소화 원칙을 적용하며, 앱 설치 직후 일괄 권한 요청을 금지한다.

## 2. 권한 최소화 원칙

| 원칙           | 확정 기준                                                                      |
| -------------- | ------------------------------------------------------------------------------ |
| 기능 기반 요청 | 권한이 필요한 기능을 사용자가 직접 선택한 시점에 요청한다.                     |
| 사전 안내      | OS 권한 팝업 전 앱 자체 안내 화면을 먼저 제공한다.                             |
| 거부 가능성    | 권한 거부 시에도 핵심 급여관리 기능은 유지한다.                                |
| 재요청 제한    | 같은 권한에 대한 반복 요청은 사용자의 명시적 재시도 시에만 한다.               |
| 대체 수단      | 사진/파일 접근은 가능한 경우 시스템 picker를 사용해 전체 저장소 권한을 피한다. |
| 플랫폼 고지    | Google Play Data safety, App Store Privacy Nutrition Label과 일치해야 한다.    |

## 3. 권한 목록

| 권한      | Android                               | iOS                            | 사용 기능                                         | 요청 시점                           | 필수 여부        |
| --------- | ------------------------------------- | ------------------------------ | ------------------------------------------------- | ----------------------------------- | ---------------- |
| 알림      | POST_NOTIFICATIONS                    | UserNotifications              | 급여일, 예산 초과, 목표 달성, 레벨업, 이벤트 푸시 | 알림 설정 ON 또는 첫 알림 활성화 시 | 선택             |
| 사진 선택 | Photo Picker / READ_MEDIA_IMAGES 제한 | Photos Picker / Limited Photos | 커뮤니티 이미지 첨부, 프로필 이미지               | 사용자가 이미지 첨부 선택 시        | 선택             |
| 파일 선택 | Storage Access Framework              | UIDocumentPicker               | 게시글 파일 첨부                                  | 사용자가 파일 첨부 선택 시          | 선택             |
| 카메라    | CAMERA                                | Camera                         | 직접 촬영 첨부                                    | 사용자가 촬영 첨부 선택 시          | 선택             |
| 생체 인증 | BiometricPrompt                       | Face ID/Touch ID               | 앱 잠금 해제, 재무 화면 보호                      | 사용자가 앱 잠금 설정 시            | 선택             |
| 네트워크  | INTERNET                              | 기본 네트워크                  | API 통신                                          | 앱 기능 전반                        | 필수 시스템 권한 |

## 4. 요청하지 않는 권한

| 권한            | 미요청 사유                                                  |
| --------------- | ------------------------------------------------------------ |
| 위치            | 급여관리·가계부 핵심 기능에 필요하지 않음                    |
| 연락처          | 친구 초대/주소록 기능이 없으므로 미수집                      |
| SMS             | 문자 기반 자동 가계부 입력은 MVP 범위가 아니며 민감도가 높음 |
| 통화 기록       | 서비스 목적과 무관                                           |
| 마이크          | 음성 기능이 없으므로 미요청                                  |
| 전체 파일 접근  | 시스템 picker로 대체                                         |
| 백그라운드 위치 | 서비스 목적과 무관                                           |

## 5. 권한별 상세 정책

### 5.1 알림 권한

| 항목            | 기준                                                                             |
| --------------- | -------------------------------------------------------------------------------- |
| 요청 시점       | 사용자가 알림 설정을 켜거나 첫 급여일/예산 초과 알림을 활성화할 때               |
| 사전 안내 문구  | `급여일, 예산 초과, 목표 달성, 레벨업 미션을 놓치지 않도록 알림을 보내드릴게요.` |
| OS 팝업 전 버튼 | `알림 허용하기` / `나중에 할게요`                                                |
| 거부 시 처리    | 앱 내 알림함은 유지, 푸시 발송 중지                                              |
| 재요청 기준     | 사용자가 설정 > 알림에서 직접 다시 켜는 경우                                     |
| 필수/선택       | 선택                                                                             |

### 5.2 사진 권한

| 항목           | 기준                                                              |
| -------------- | ----------------------------------------------------------------- |
| 요청 시점      | 프로필 이미지 변경 또는 커뮤니티 글쓰기에서 이미지 첨부 선택 시   |
| 기본 방식      | 전체 갤러리 접근이 아닌 시스템 사진 선택기 우선                   |
| 사전 안내 문구 | `게시글 또는 프로필에 첨부할 사진만 선택해서 사용할 수 있습니다.` |
| 거부 시 처리   | 이미지 없이 글 작성 가능                                          |
| 저장           | 선택된 이미지 파일만 업로드, 원본 EXIF 위치정보 제거              |

### 5.3 파일 권한

| 항목           | 기준                                                                           |
| -------------- | ------------------------------------------------------------------------------ |
| 요청 시점      | 커뮤니티 글쓰기에서 파일 첨부 선택 시                                          |
| 기본 방식      | 시스템 파일 선택기 사용                                                        |
| 사전 안내 문구 | `게시글에 첨부할 파일만 선택합니다. 선택하지 않은 파일에는 접근하지 않습니다.` |
| 제한           | 1개 파일 최대 10MB, 허용 확장자 jpg/png/webp/pdf/txt                           |
| 거부 시 처리   | 파일 없이 글 작성 가능                                                         |

### 5.4 카메라 권한

| 항목           | 기준                                                                 |
| -------------- | -------------------------------------------------------------------- |
| 요청 시점      | 사용자가 `사진 촬영` 버튼을 누른 경우                                |
| 사전 안내 문구 | `게시글 또는 프로필에 사용할 사진을 직접 촬영할 수 있습니다.`        |
| 거부 시 처리   | 사진 선택 또는 첨부 없이 진행 가능                                   |
| 저장           | 촬영 결과물만 앱 임시 저장 후 업로드, 미등록 임시파일 24시간 내 삭제 |

### 5.5 생체 인증 권한

| 항목           | 기준                                                                           |
| -------------- | ------------------------------------------------------------------------------ |
| 요청 시점      | 사용자가 `앱 잠금 사용`을 켜는 경우                                            |
| 목적           | 급여/지출/저축 화면 보호                                                       |
| 사전 안내 문구 | `급여와 지출 정보를 더 안전하게 보호하기 위해 생체 인증을 사용할 수 있습니다.` |
| 거부 시 처리   | PIN 또는 비밀번호 재인증 사용                                                  |
| 저장           | 생체정보는 앱 서버에 저장하지 않으며 OS 보안영역을 사용                        |

## 6. 권한 거부/철회 대응

| 상황           | 앱 처리                                                      |
| -------------- | ------------------------------------------------------------ |
| 알림 거부      | 푸시 미발송, 앱 내 알림함 유지, 설정 진입 안내               |
| 사진 거부      | 첨부 버튼 비활성화 대신 시스템 설정 안내, 텍스트 글쓰기 유지 |
| 파일 거부      | 파일 첨부 생략, 게시글 작성 유지                             |
| 카메라 거부    | 사진 선택 대체 안내                                          |
| 생체 인증 거부 | PIN/비밀번호 인증 대체                                       |

## 8. 최종 완료 선언

본 권한 정책서는 알림, 사진, 파일, 카메라, 생체 인증, 네트워크 권한의 요청 시점, 문구, 거부 처리, 대체 흐름, 스토어 고지 기준을 모두 포함한다. 따라서 본 문서는 문서상·이론상 더 이상 수행할 작업이 없는 최종본으로 확정한다.
