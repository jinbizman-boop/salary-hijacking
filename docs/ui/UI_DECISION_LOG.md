## Intentional Differences Allowed In Visual Tests

- Korean product copy may differ from Stitch demo copy when Stitch contains temporary names, wrong dates, or non-product wording.
- Accessibility labels, privacy labels, and ad disclosures may add text not visible in raw Stitch screenshots.
- Server/error/offline states may use product-safe copy instead of visual-demo placeholders.
- Logo rendering uses committed brand assets, not screenshots as backgrounds.
