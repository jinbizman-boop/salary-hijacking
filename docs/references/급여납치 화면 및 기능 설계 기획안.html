월급이 사라지기 전에
먼저 붙잡아두는 급여 자기관리 앱

급여납치는 월급 수령 직후 고정지출·저축·생활비·일일 소비를 먼저 분리하고, 남은 금액을 “납치금액”으로 시각화해 사용자의 소비 통제와 자기계발 루틴을 함께 관리하는 모바일 앱이다.

핵심 목적

월급 흐름을 하루 단위로 가시화하고 계획 소비를 유도

주 사용자

사회초년생, 직장인, 소비 통제가 필요한 월급 생활자

차별 포인트

급여 관리 + 레벨업 + 인증 커뮤니티 결합

기획 기준

스플래시부터 마이페이지까지 총 17개 화면 반영

출처 및 적용 기준
본 문서는 첨부 파일 「급여납치 어플리케이션 디자인.pdf」의 1~17페이지 화면을 기준으로 재구성했다. 주요 반영 화면은 스플래시/로그인, 급여 현황, 알림, 급여 계획, LV UP, 독서·뉴스·영어·건강 레벨업, 커뮤니티, 글쓰기, 마이페이지다.

2. 기획 방향

Core Value월급 선점 관리

월급을 받은 뒤 남는 돈을 보는 것이 아니라, 먼저 지킬 돈을 정하고 하루 소비를 통제한다.

Behavior Design납치금액 성과화

저축·예산 잔여분·지출 통제를 “납치금액”으로 표현해 성취감을 만든다.

Retention매일 레벨업 루틴

독서·뉴스·영어·운동 미션으로 앱 접속 빈도를 높이고 금융 관리의 습관화를 돕는다.

서비스 포지셔닝

구분 기존 가계부 앱 급여납치 앱
핵심 질문 얼마를 썼는가? 이번 월급에서 얼마를 지켜냈는가?
주요 지표 지출, 카테고리, 월간 합계 수령금액, 지출금액, 납치금액, 목표 달성률, 레벨
사용 방식 소비 후 기록 급여 수령 전후 계획 → 일일 예산 관리 → 성과 확인
확장 기능 통계, 카드 연동 자기계발 콘텐츠, 인증 커뮤니티, 포인트 이벤트, 광고/제휴
서비스 이름의 “납치”는 강한 후킹 장점이 있지만 부정적 인상도 있을 수 있다. 온보딩에서는 “월급을 사라지기 전에 내가 먼저 확보한다”는 긍정적 의미를 반드시 설명해야 한다.

대표 계산 구조

예상 납치금액 = 수령 예정 급여 - 지출 예정 금액
오늘 남은금액 = 일일 설정금액 - 오늘 사용금액
누적 납치금액 = 기존 누적 납치금액 + 월별 확정 납치금액
목표 달성률 = 총 누적 납치금액 ÷ 이번 달 목표 납치금액 또는 설정 목표값 × 100

운영 정책

광고 정책

금융성 정보와 광고 추천은 명확히 분리한다. 사용자의 급여·대출·저축 데이터 기반 민감 타겟팅은 별도 동의가 필요하다.

커뮤니티 정책

월급, 부업, 투자, 건강 주제는 오정보 위험이 있으므로 신고, 숨김, 운영자 검수 기준이 필요하다.

알림 정책

결제 예정/예산 초과/목표 달성 알림은 기본 제공하되, 콘텐츠 추천 알림은 사용자가 끌 수 있어야 한다.

7. 작동형 UI/UX 예시안

오른쪽 모바일 프레임은 실제로 클릭·입력·전환이 가능한 HTML 프로토타입이다. 로그인, 하단 탭 이동, 지출 추가, 계획 수정, 레벨업 완료, 커뮤니티 글쓰기 등록이 작동한다.

프로토타입 사용 방법

1. 로그인

아무 소셜 버튼이나 누르면 홈 화면으로 이동한다.

2. 지출 추가

홈 화면에서 금액을 입력하고 “추가”를 누르면 오늘 사용금액과 남은금액이 바뀐다.

3. 계획 변경

계획 탭에서 급여·지출·목표금액을 수정하면 달성률과 납치금액이 재계산된다.

4. 커뮤니티 작성

커뮤니티 탭의 초록색 작성 버튼을 누르면 게시글 작성 화면이 열리고 등록된다.

8. MVP 및 개선 로드맵

단계 범위 핵심 산출물 성공 기준
1차 MVP 로그인, 급여 홈, 계획 설정, 일일 예산, 지출 기록 월급 관리 핵심 루프 사용자가 3분 안에 급여 계획을 완성하고 오늘 예산을 확인
2차 고도화 알림, 목표 달성률, 레벨업 콘텐츠 습관 형성 루틴 주 3회 이상 재방문, 미션 완료율 측정
3차 커뮤니티 게시판, 인증, 좋아요/댓글/공유, 신고 직장인 자기관리 커뮤니티 사용자 생성 콘텐츠와 인증글 축적
4차 수익화 광고, 제휴, 포인트 이벤트, 추천 콘텐츠 광고/제휴 운영 구조 광고 피로도 관리와 클릭/전환 로그 확보

우선 개선 제안

기능 범위 정리

초기 버전은 급여 관리와 지출 통제에 집중하고, 레벨업은 보조 기능으로 배치한다.

입력 부담 축소

지난달 복사, 추천 템플릿, 빠른 지출 버튼을 제공해 사용자가 오래 입력하지 않게 한다.

신뢰 장치 강화

급여·대출·저축 정보 보호 안내, 광고 분리 원칙, 개인정보 처리 동의를 명확히 보여준다.
